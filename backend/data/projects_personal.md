# Personal Projects

## Meet Miriam AI

Meet Miriam AI is my flagship personal project.

The goal is to create an AI-powered portfolio where recruiters can have a real conversation with an AI assistant that represents me professionally.

Current technologies:

- React
- TypeScript
- FastAPI
- OpenAI API

Current architecture:

- Context Builder
- Prompt Builder
- Provider Pattern
- Modular Backend
- Markdown Knowledge Base

Planned roadmap:

- RAG
- Conversation Memory
- Streaming Responses
- Vector Database
- Resume Ingestion

This project demonstrates both software engineering skills and my ability to quickly learn new technologies.
